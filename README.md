<div align=center>
  <h1>Dirjson</h1>
  <h6>Converts a directory strucutre into json.</h6>
</div>

## Installation

```shell
npm install dirjson --save
```

## Usage

```shell
Dirjson dir/
```

## Output
```javascript
{
  "file1": "file1.txt",
  "folder": {
    "file2": "folder/file2.txt",
    "file3": "folder/file3.txt"
  }
}
```
